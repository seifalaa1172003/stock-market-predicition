# stock-market-predicition
Stock Price Movement Predictor – ML system forecasting up/down/flat moves using AMZN, DPZ, BTC, NFLX data. Features 4 models (Logistic Regression, Random Forest, Decision Tree, SVM) with Tkinter GUI. Includes accuracy, precision, CV scores &amp; confusion matrix visualization.
Key Features
Multiple ML Models – Choose between:

    Logistic Regression

    Random Forest

    Decision Tree

    Support Vector Machine (SVM)

Interactive GUI – Easy-to-use interface for entering stock prices and getting instant predictions

Model Performance Metrics:

    Accuracy Score

    Weighted Precision Score

    5-Fold Cross-Validation Scores

    Mean Cross-Validation Score

    Confusion Matrix visualization

Data Preprocessing:

    Automatic handling of missing values

    Feature standardization (StandardScaler)

    Label encoding for price movements
